A：内部文档流元素高度的总和

Q：文档流是什么？
A：文档内元素流动方向（内联元素从左往右，宽度不够换行；块级元素每个块级元素占一行，从上往下流动；）

Q：如何设置块级元素文档流从左到右？
A：

  1. display:inline-block;
  2. float:left;清除父级元素浮动。（推荐）

Q：什么是空元素？
A：不能存在子节点的元素，如<input>、<meta>、<link>元素。

Q：什么是伪元素？
A：
伪元素是某个元素的某个部分，需要通过CSS选择器来选择匹配并设置样式，比如.box::before表示class名为box的元素的第一个子元素，默认伪元素不显示，需要设置其CSS属性content来让其显示。
常见的伪元素有::before，::after，默认是一个行内元素，可以自行修改，一个简单的使用效果：点我试试。

Q：什么是伪类？
A：伪类是添加到CSS选择器上的关键字,用来指定要选择元素的特殊状态。比如:hover就是指鼠标悬浮在某元素上时所使用的样式，例如：.box:hover{ background-color:#fff }表示当鼠标移到class名为box的元素上时，该元素的背景颜色为白色（#fff）。

Q：如何消除CSS属性display值为inline-block的元素的底部边距？
A：设置CSS属性vertical-align:top。
